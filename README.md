# AI-ML-Tasks

Simple project with 3 tasks:

- `Task-1-Iris` - dataset exploration and visualization
- `Task-2-Heart` - heart disease prediction
- `Task-3-HealthBot` - health query chatbot

## Quick Run Commands

Run from project root (`AI-ML-Tasks`):

```powershell
python .\run_task1.py
python .\run_task2.py
python .\run_task3.py
python .\run_task3.py --chat
```

## Notes

- First run may take time (creates `.venv` and installs dependencies).
- For chatbot, set provider keys in `Task-3-HealthBot/.env`.
